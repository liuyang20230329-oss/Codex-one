# 37° 测试用例

- 文档版本: `2026.03.29-r1`
- 对应版本: `0.5.0-dev.1+2026032901`

## 1. 登录与注册

### 用例 1: 演示账号登录
- 前置条件: App 未登录
- 步骤:
  1. 打开登录页
  2. 输入 `demo@codex.one`
  3. 输入 `Password123!`
  4. 点击 `登录`
- 预期结果:
  - 成功进入首页
  - 顶部显示 `37°`
  - 底部显示 `广场 / 圈子 / 消息 / 我的`

### 用例 2: 新用户注册需要勾选协议
- 前置条件: App 未登录
- 步骤:
  1. 切换到 `注册`
  2. 输入昵称、邮箱、密码、确认密码
  3. 不勾选协议，点击 `创建账号`
- 预期结果:
  - 页面提示 `请先同意用户协议与隐私政策。`
  - 不会创建账号

### 用例 3: 新用户注册成功
- 前置条件: App 未登录
- 步骤:
  1. 切换到 `注册`
  2. 输入有效昵称、邮箱、密码
  3. 勾选协议
  4. 点击 `创建账号`
- 预期结果:
  - 自动进入首页
  - “我的”页显示新账号默认资料

## 2. 账号与认证

### 用例 4: 手机号认证成功
- 前置条件: 已登录
- 步骤:
  1. 进入 `我的`
  2. 点击 `手机号认证`
  3. 输入 11 位手机号
  4. 获取验证码并填写演示验证码
  5. 提交
- 预期结果:
  - 显示 `手机号认证已完成`
  - 认证中心对应项变为 `已认证`
  - 聊天私聊权限开启

### 用例 5: 未完成手机号认证时私聊受限
- 前置条件: 已登录但未完成手机号认证
- 步骤:
  1. 进入 `消息`
  2. 打开任一私聊会话
  3. 尝试输入并发送消息
- 预期结果:
  - 输入区被限制
  - 页面提示 `请先完成手机号认证后再开始私聊`
  - 消息不会发送

### 用例 6: 未完成手机号认证时系统引导会话可用
- 前置条件: 已登录但未完成手机号认证
- 步骤:
  1. 进入 `消息`
  2. 打开 `37° 向导`
  3. 发送一条消息
- 预期结果:
  - 消息发送成功
  - 系统回复正常返回

### 用例 7: 更换头像后重置本人认证
- 前置条件: 已完成实名和本人认证
- 步骤:
  1. 进入 `我的`
  2. 编辑头像主题
  3. 保存
- 预期结果:
  - 本人头像认证状态重置
  - 页面提示需重新完成人脸/头像认证

## 3. 我的页面

### 用例 8: 资料完成度联动
- 前置条件: 已登录
- 步骤:
  1. 依次补充性别、出生年月、个性签名、视频介绍、作品
  2. 观察顶部资料完成度
- 预期结果:
  - 完成度百分比持续提升
  - 资料完成度进度条同步变化

### 用例 9: 作品新增与删除
- 前置条件: 已登录
- 步骤:
  1. 进入 `我的作品`
  2. 添加一条语音/视频/图片作品
  3. 再删除该作品
- 预期结果:
  - 新增后列表立即可见
  - 删除后列表即时更新
  - 系统向导会话收到资料变更提醒

## 4. 广场与圈子

### 用例 10: 广场筛选
- 前置条件: 已登录
- 步骤:
  1. 进入 `广场`
  2. 依次切换地区、年龄、性别筛选
- 预期结果:
  - 用户列表随筛选条件变化
  - 无匹配时显示空状态提示

### 用例 11: 圈子发布基础动态
- 前置条件: 已登录
- 步骤:
  1. 进入 `圈子`
  2. 点击右下角 `发布动态`
  3. 填写文案、定位，并勾选一种素材类型
  4. 点击 `立即发布`
- 预期结果:
  - 新动态插入列表顶部
  - 卡片显示定位、认证状态、互动数和附件标签
