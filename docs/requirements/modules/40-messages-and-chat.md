# 模块四：消息与聊天

## 1. 模块目标

消息模块负责关系入口，会话模块负责真正聊天。

## 2. 页面结构建议

按你当前要求，消息页分成两层。

1. 顶部关系分组
2. 下部聊天会话列表

## 3. 顶部分组模板

固定分组:

- 好友
- 热聊的
- 关注我的
- 我关注的

### 需要你补充

- 每个分组点击后去哪里:
- 是否显示人数:
- 是否显示红点:
- 是否支持查看更多:

## 4. 会话列表模板

- 卡片展示字段:
- 最后一条消息是否显示:
- 时间格式:
- 未读红点样式:
- 是否支持置顶:
- 是否支持删除:

## 5. 聊天详情模板

- 顶部显示:
- 中间消息流:
- 底部输入区:
- 是否支持文字:
- 是否支持表情:
- 是否支持图片:

- 是否支持视频入口:

## 6. 消息状态模板

- 发送中:
- 发送成功:
- 发送失败:
- 未读:
- 已读:

## 7. 需要你线下补充的重点

- 谁可以发起聊天
- 陌生人能否直接私聊
- 未认证用户发消息限制
- 红点清除规则
- 会话排序规则
