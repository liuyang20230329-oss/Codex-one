# 模块六：公共能力

## 1. 模块目标

这个模块不直接面向一个页面，但会影响全部页面实现。

## 2. 当前建议包含的公共能力

- 底部导航
- 页面路由
- 上传和媒体选择
- 权限申请
- 系统通知
- 审核状态
- 本地缓存
- 错误提示
