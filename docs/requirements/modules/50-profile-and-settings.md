# 模块五：我的与设置

## 1. 模块目标

这个模块负责用户展示自己、编辑自己、管理自己。

## 2. 页面结构建议

建议分成四块。

1. 顶部个人信息卡
2. 中部资料完善区
3. 作品展示区
4. 底部设置区

## 3. 个人信息模板

你已经明确的资料项如下。

- 头像
- 性别
- 年龄
- 出生年月
- 个性签名
- 视频介绍
- 我的作品

### 每个字段需要你补充

- 是否必填:
- 展示优先级:
- 编辑方式:
- 是否公开:
- 是否需要审核:

## 4. 视频介绍模板

- 最长时长:
- 封面是否必填:
- 是否自动播放:
- 是否审核后显示:

## 5. 我的作品模板

支持类型:

- 语音
- 视频
- 图片

### 需要你补充

- 每种类型上限:
- 展示顺序:
- 是否允许置顶:
- 是否允许删除:

## 6. 设置区模板

- 账号安全:
- 通知设置:
- 黑名单:
- 清理缓存:
- 退出登录:
- 注销账号是否本期做:

## 7. 需要你线下补充的重点

- 哪些资料默认公开
- 哪些资料只对好友可见
- 作品是否参与广场和圈子曝光
- 是否要加资料完成度进度条
