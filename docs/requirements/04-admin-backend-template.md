## 6. 审核流转模板

### 6.1 实名认证审核流

```text
用户提交 -> 待审核 -> 审核通过 / 驳回 -> 通知用户 -> 记录审核日志
```

### 6.2 本人认证审核流

```text
用户上传头像或人脸材料 -> 待审核 -> 审核通过 / 驳回 -> 更新用户认证状态 -> 通知用户
```

### 6.3 内容审核流

```text
用户发布内容 -> 待审核 / 先发后审 -> 审核通过 / 驳回 / 下线 -> 通知用户 -> 保留审核记录
```
